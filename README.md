## ⚡ 特性
- 🛠 快速简单地为您的博客创建封面图片
- ✨ 15+ 不同的背景图案
- 💾 以 `.png` 或 `.jpeg` 格式下载封面图片

## 👩‍💻 开发
这个项目是用 [Create React App](https://github.com/facebook/create-react-app) 开发的。



```shell
git clone https://github.com/thedavidweng/CoverView-CN.git
cd CoverView/
npm start
```
